树一树习惯小程序 为您解说

项目简介

一个通过线上模拟树屋商城方式，帮助习惯养成的微信小程序
参加腾讯高校微信小程序开发大 赛东北赛区二等奖，互联网+创新创业大赛黑龙江省三等奖等，荣获四个奖项
树一树便是帮助用户坚持习惯养成。挖掘用户喜欢的东西，通过美观的房屋精灵吸引用户，并在坚持养成习惯的过程中给予丰富奖励与鼓励，趣味无穷

项目职责

本人担任项目队长
从想法创意，需求分析到学习微信小程序开发，书写计划书视频，原型 UI 界面 设计，比赛答辩均由本人完成
个人完成该项目前端代码，前端交互，与后端连接。
后端，数据库设计,ui设计由队员帮助完成

项目技术

使用 wxml,wxss,实现前端界面，使用 js 调用 API 实现页面交互
使用 axure 实现原型,使用ps进行ui设计
设用腾讯云平台托管后端数据库 后端整体使用 PHP 的 Codelgniter 框架，使用 git 进行团队
